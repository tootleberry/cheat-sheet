Cheat Sheet

HTML Table References

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The data in a table cell
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Tag Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

CSS Selectors

Selector Example Example Description
element p Selects all <p> elements
#id #name Selects the element with id="name"
element * Selects all elements
.class .intro
p.intro		 Selects all elements with class="intro"
Selects all <p> elements with class="intro"

CSS References

Property Value Description
background background-color Specifies the background color to be used
background-img Specifies ONE or MORE background images to be used
display inline Displays an element as an inline element (like <span>). Any height and width properties will have no effect. This is default.
block Displays an element as a block element (like <p>). It starts on a new line, and takes up the whole width
inline-block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values
font-family name Required. Defines the name of the font.
src url() Required. Defines the URL(s) where the font should be downloaded from
